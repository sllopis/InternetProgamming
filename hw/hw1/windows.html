Windows: The birth of Technology

The giant tech company Microsoft created the platform Windows in the mid 1980s. Over the years, there have been
many updates released by Windows, being the most recent ones:
  • Windows 10 (2015)
  • Windows 8 (2012)
  • Windows 7 (2009)
  • Windows Vista (2007)
Windows is the most famous and used Operating System in the world since
most brand-new computers have it already incorporated.
