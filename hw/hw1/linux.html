Get creative with Linux

Linux (pronounced LINN-ux) is an open source Operating System, which means they can be
modified and distributed by anyone around the world. This makes it different to other propietary software
such as Windows, which only Microsoft can modify its own product. The best about Linux is that a free
resource to download, and there are many versions and options to choose from. The software is customizable
by users.
